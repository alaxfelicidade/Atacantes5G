# Atacantes 5G — PWA

## Como publicar gratuitamente (5 minutos)

### Opção 1 — Netlify (mais fácil, arrasta e solta)
1. Acesse **netlify.com** e crie uma conta grátis
2. Na tela inicial clique em **"Add new site" → "Deploy manually"**
3. **Arraste a pasta inteira** `pwa_atacantes` para a área indicada
4. Pronto! Você receberá um link tipo `https://atacantes5g.netlify.app`

### Opção 2 — GitHub Pages (também grátis)
1. Crie conta em **github.com**
2. Novo repositório → faça upload dos arquivos desta pasta
3. Settings → Pages → Branch: main → Save
4. Link gerado: `https://seuusuario.github.io/atacantes5g`

---

## Como instalar no celular

### Android (Chrome)
- Acesse o link pelo Chrome
- Aparecerá banner automático na tela → toque **"Instalar"**
- O app aparece na tela inicial como qualquer app

### iPhone (Safari)
- Acesse o link pelo Safari
- Toque no ícone de **Compartilhar** (quadrado com seta)
- Toque em **"Adicionar à Tela de Início"**
- O app aparece na tela inicial com o ícone dos Atacantes 5G

## Arquivos incluídos
- `index.html` — o app completo
- `manifest.json` — configurações do PWA
- `sw.js` — service worker (modo offline)
- `icon-512.png` — ícone 512×512
- `icon-192.png` — ícone 192×192
- `icon-180.png` — ícone Apple Touch
- `icon-32.png` — favicon
